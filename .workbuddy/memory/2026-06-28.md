# 2026-06-28 工作日志

## ProseMirrorEditor 输入时 Delete/Backspace 导致节点被删 — 修复

**文件:** `packages/canvas-core/src/nodes/image/ImageBottomToolbar.vue`

**问题:** 在图片节点底部工具栏的 ProseMirrorEditor 中输入时按 Delete/Backspace 会触发 VueFlow 删除当前选中节点。

**根因:** 
- VueFlow 内部监听 Backspace/Delete 键删除选中节点
- ProseMirror 编辑器是 contentEditable div，不是 HTMLInputElement/HTMLTextAreaElement
- ShortcutManager 的输入框跳过逻辑（`e.target instanceof HTMLInputElement || HTMLTextAreaElement`）无法匹配 ProseMirror 的 contentEditable div
- 键盘事件从编辑器冒泡到 VueFlow 的 keydown 处理器，触发了节点删除

**修复:** 
- 在 `.editor-wrapper` div 上添加 `@keydown="onEditorKeydown"`
- `onEditorKeydown` 对 `Delete` 和 `Backspace` 键调用 `e.stopPropagation()` 阻止冒泡
- ProseMirror 编辑器自身仍正常处理这两个键（删除文本），只是事件不再冒泡到外层
