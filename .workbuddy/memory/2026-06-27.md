## UiTestPage.vue TS2307 模块解析修复

**问题**: `vue-tsc --project tsconfig.app.json` 报:
- `TS2307: Cannot find module '@mini-canvas/canvas-core/components/Ui'`
- `TS2307: Cannot find module '@mini-canvas/canvas-core/components/Ui/types'`

**根因**: `canvas-core/package.json` exports 野卡 `"./*": "./src/*"` 在 TS strict project mode 无法解析目录型子路径。

**修复**: 添加显式 subpath exports + `types` 字段:
```json
"types": "./src/index.ts",
"exports": {
  ".": { "types": "./src/index.ts", "default": "./src/index.ts" },
  "./components/Ui": { "types": "./src/components/Ui/index.ts", "default": "./src/components/Ui/index.ts" },
  "./components/Ui/types": { "types": "./src/components/Ui/types.ts", "default": "./src/components/Ui/types.ts" },
  "./*": "./src/*"
}
```
验证: `vite build` 通过，`vue-tsc --project tsconfig.app.json` 中 UiTestPage 错误已消除。
