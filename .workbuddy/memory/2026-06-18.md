## 13:09 — CustomEdge: 修复左侧端口连接线方向问题

**Bug**: `CustomEdge.vue` 贝塞尔曲线计算用了 `Math.abs(targetX - sourceX) * 0.5`，控制点始终向右偏移。当 source 在 target 右侧时，曲线从 target 右侧绕入左侧端口，方向反了。

**Fix**: `CustomEdge.vue:49,78` — 去掉 `Math.abs()`，改为 `(targetX - sourceX) * 0.5`，让控制点方向跟随 source→target 实际位置关系。

- source 在 target 左边 → dx > 0 → 曲线向右 → 从左侧接近 target ✓
- source 在 target 右边 → dx < 0 → 曲线向左 → 从右侧接近 target ✓

**Verification**: vue-tsc passed. 1 file, 2 lines.
