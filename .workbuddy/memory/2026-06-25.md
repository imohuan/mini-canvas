# 2026-06-25

## Bug Fix: 连线拖拽创建节点时输出端口丢失

**根因**: `ContextMenuPlugin.ts` 第409行，connection 模式创建节点时 `requireSource: isReverse`。
正向连线(sourceHandle="source", isReverse=false)时，`requireSource=false` 导致 `createNode` 中的 `options.requireSource !== false` 判断失败，`sourcePosition` 被跳过，新节点丢失输出端口。

**修复**: 移除 `requireSource` 参数，只保留 `requireTarget`。输出端口由节点类型定义决定，不应被连线创建上下文覆盖。修改位置: `ContextMenuPlugin.ts:409`。
