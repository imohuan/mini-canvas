## Canvas + VueFlow 双数据绑定 zoom-out 分析
- 全景分析了 Pinia Store 与 VueFlow 内部状态的双数据绑定架构
- 约 30+ 处同步逻辑代码点，核心适配层在 PluginContext.ts createActions 工厂

## 修复 custom-node-frame z-index 遮盖问题
- `BaseNode.vue` 第 260 行: `z-index: 0` → `z-index: -1`
- 原因: position:absolute + z-index:0 的定位元素绘制在正常流内容之上，导致 background layer 遮盖图片/视频
